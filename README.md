# ToDoList

Todo List приложение на Vanilla JavaScript с использованием собственного micro-framework `elmish.js`.

## Стек

- HTML5
- CSS3
- Vanilla JavaScript
- Elm architecture style
- localStorage

## Возможности

- Добавление задач
- Удаление задач
- Редактирование задач
- Отметка выполненных
- Фильтрация задач
- Очистка completed
- Сохранение в localStorage

## Запуск

Просто откройте `index.html` в браузере.

## Архитектура

Используется упрощённая Elm architecture:

- state
- update
- render
- subscriptions
